🚂 Mocăniță 3D – OpenGL Project

🧠 About the Project

This project is a 3D scene built using OpenGL, representing a simplified steam locomotive (“mocăniță”).

It demonstrates core concepts of computer graphics, including:

geometric modeling
transformations in 3D space
lighting
animation
shadow projection

🔧 Features
🚀 3D Modeling
Built using cubes and cylinders
Includes:
chassis
cabin
boiler
chimney
wheels

🔄 Animation
Wheels rotate continuously using an angle updated in IdleFunction
Scene rotation controlled with keyboard input (X)

💡 Lighting
Implemented using GL_LIGHT0
Enhances depth and realism of the model

🌑 Shadow Projection
Shadow is projected on a ground plane
Implemented using a shadow matrix
Simulates realistic light behavior

🎮 Interaction
Press X → rotate the entire scene

🖥️ Technologies Used
🧩 OpenGL (GL, GLU, GLAUX)
💻 C / C++
🪟 Windows API
