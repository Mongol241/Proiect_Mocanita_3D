Mocăniță 3D – OpenGL Project

This project represents a 3D scene built using OpenGL, featuring a simplified model of a steam locomotive (“mocăniță”). The scene demonstrates fundamental concepts of computer graphics, including object modeling, transformations, lighting, animation, and shadow projection.

🔧 Features
3D Modeling
The locomotive is constructed using basic primitives such as cubes and cylinders, combined through translation, rotation, and scaling.
Animation
The wheels are animated using a continuous rotation, simulating real locomotive motion.
Lighting
A light source is added to enhance depth perception and provide realistic shading.
Shadow Projection
The object casts a shadow onto a ground plane using a shadow projection matrix.
Interaction
Press X to rotate the entire scene.
